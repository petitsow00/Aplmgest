🔄 Réinitialiser les données d'une entreprise

Supprime toutes les données opérationnelles d'une entreprise (clients, dossiers, caisse, factures…)
mais conserve les utilisateurs et les paramètres de l'entreprise.

⛔ Réinitialisation complète de la plateforme

Supprime toutes les entreprises, toutes leurs bases de données, et tout le journal d'activité.
Seul le compte Super Administrateur est conservé. La plateforme revient à l'état initial.

 Comment publier une mise à jour ?

        

          
    
            
  1. Faites vos modifications dans le code de l'application
    2. 
            
  2. Modifiez le numéro de version dans package.json (ex: 1.0.1)
    3. 
            
  3. Reconstruisez l'installateur : npm run build:win
    4. 
            
  4. Hébergez le nouveau fichier .exe en ligne (Google Drive, Dropbox, GitHub…)
    5. 
            
  5. Copiez le lien de téléchargement direct et remplissez le formulaire ci-dessus
    6. 
            
  6. Cliquez Générer le JSON, copiez-le et mettez à jour votre fichier version.json en ligne
    7. 
            
  7. Vos clients verront une notification de mise à jour disponible
    8. 
          

          

            Conseil gratuit : Utilisez GitHub Gist — créez un fichier version.json,
            cliquez sur le bouton "Raw" et copiez cette URL ici. C'est gratuit et instantané.
